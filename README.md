# Customer-segmentation
●	Analysed customer demographic and transactional datasets to identify behavioural patterns 
●	Performed data cleaning, preprocessing, and feature selection to prepare data for analysis 
●	Conducted exploratory data analysis (EDA) to understand customer spending and engagement trends 
●	Applied clustering techniques to segment customers based on behavioural similarities 
●	Interpreted customer segments to identify high-value customers, low-engagement users, and churn-risk groups 
●	Created visualisations to compare customer behaviour across different segments 
●	Developed an interactive Streamlit dashboard to enable dynamic exploration of customer data and insights
